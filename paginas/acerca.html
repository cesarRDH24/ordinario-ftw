Acerca de MusicHub

Descripción

MusicHub es un sitio web desarrollado como proyecto final de la materia Fundamentos de Tecnologías Web.

El objetivo es presentar información relacionada con artistas, álbumes, canciones y géneros musicales utilizando HTML, CSS, JavaScript y XML.

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • XML
  • Git y GitHub
